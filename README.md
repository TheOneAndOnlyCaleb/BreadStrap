# Breadstrap 🍞

**Breadstrap** is a lightweight, efficient FastFlag (FFlag) injector for Roblox. Designed with simplicity in mind, it allows users to easily modify client-side settings, unlock performance features, and experiment with hidden engine toggles without the bloat of traditional injectors.

---

## 🚀 Features

* **FastFlag Injection:** Instantly apply FFlags to customize your Roblox client.
* **Performance Optimization:** Unlock higher framerates and reduce latency with pre-configured presets.
* **Minimalist GUI:** A clean, "no-nonsense" interface for quick adjustments.
* **Lightweight:** Low memory footprint, ensuring your game remains the priority.
* **Safe & Local:** Operates on local configuration files to ensure maximum stability.

---

## 📦 Installation

1.  **Download:** Navigate to the [Releases](https://github.com/TheOneAndOnlyCaleb/BreadStrap/releases/tag/bread) page and download the latest `.exe`
2.  **Extract:** Unzip the files to a folder of your choice.
3.  **Run:** Launch `Breadstrap.exe` as an administrator to ensure it has the permissions to modify client settings.

---

## 🛠 Usage

1.  Open **Breadstrap**.
2.  Enter the FFlags you wish to apply or select from the built-in list.
3.  Click **Apply Settings**.
4.  Launch (or restart) Roblox for the changes to take effect.

> **Note:** If Roblox updates, you may need to re-apply your flags depending on the version change.
